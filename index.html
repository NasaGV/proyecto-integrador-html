Bienvenido

Este proyecto tiene como objetivo explicar los principios de la Web y la importancia de utilizar HTML5 semántico para crear páginas bien estructuradas y accesibles.

Temas principales

HTML5

HTML5 permite estructurar el contenido de una página web utilizando etiquetas que describen claramente la función de cada elemento.

Etiquetas semánticas

Las etiquetas semánticas facilitan la comprensión del código para los desarrolladores, los navegadores y las tecnologías de asistencia.

Accesibilidad

Aplicar buenas prácticas de accesibilidad permite que cualquier persona pueda navegar e interactuar con una página web de forma adecuada.
